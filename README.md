# Balanced String Algorithms Project

## Project Topic
This project solves the **Longest Balanced Substring** problem.

A balanced substring is a substring that contains exactly **two distinct characters** and both characters appear the **same number of times**.

## Implemented Algorithms
This project includes two different implementations:

1. **Non-Recursive Approach**
   - Uses nested loops.
   - Time Complexity: **O(n²)**
   - Space Complexity: **O(1)**

2. **Recursive Approach**
   - Uses recursive functions to generate and check substrings.
   - Time Complexity: **O(n³)**
   - Space Complexity: **O(n)**
